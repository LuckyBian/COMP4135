# COMP4135 Group Project

## Build Enviroment
```
conda create --name rs python=3.6
```
## Activate the Enviroment
```
source activate rs
source flask/bin/activate
```
## Install some package
```
pip install Flask
pip install Flask-Table
pip install pandas
pip install numpy
```



## Usage
```
python app.py
```
Go to `http://127.0.0.1:5000/` on your browser.
